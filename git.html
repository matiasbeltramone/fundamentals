🔧 Introducción a Git

Control de versiones para proyectos de programación

📌 ¿Qué es Git?

Git es un sistema de control de versiones que permite llevar un historial de los cambios en el código y colaborar en equipo de forma ordenada.

🛠️ Comandos básicos


git init          # Inicia un repositorio
git status        # Ver estado actual
git add .         # Agrega cambios al área de preparación
git commit -m ""  # Confirma los cambios
git push          # Sube los cambios al remoto

✅ Conclusión

Git es la base del trabajo colaborativo en desarrollo de software. Con él podemos integrar CI/CD y trabajar en equipo sin caos.
